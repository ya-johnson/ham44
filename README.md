# Ham44 - info about Lewis Hamilton

### Built with: React Js, Zustand, Recharts

### Tech

- [React Js](https://reactjs.org/)
- Routing: [React Router Dom V6](https://reactrouter.com/)
- State Managment: [Zustand](https://zustand-demo.pmnd.rs/)
- Charts: [Reachrts](https://recharts.org/)

### Installation

```
$ git clone 'https://github.com/ya-johnson/ham44.git'
$ npm install
$ npm start
```
